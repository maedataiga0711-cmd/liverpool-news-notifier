# 🔴 Liverpool News Notifier
BBC Sportからリヴァプールの最新ニュースを取得し、LINEで通知する自動巡回システムです。

## 機能
- **特定チームの自動抽出**: BBCのチーム専用ページをスクレイピング。
- **LINE通知**: Messaging API v3 を使用した最新の通知形式。
- **スケジュール実行**: `schedule`ライブラリにより、1日3回の定時通知が可能。
- **最新環境対応**: Python 3.14環境での動作確認済み。

## 開発の経緯
LINE Notifyのサービス終了に伴い、最新のMessaging APIを活用した通知システムを自作しました。
Python 3.14環境におけるライブラリの互換性警告に対処するなど、最新の技術動向を意識して開発しました。
